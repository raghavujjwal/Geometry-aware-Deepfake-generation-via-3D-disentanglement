# Logs Folder

Use this folder for raw Kaggle exports and provenance material useful for verification.

Recommended contents:

- raw copies of:
  - `main_results_partial.csv`
  - `benchmark_pairwise_results.csv`
  - `id_retrieval_results.csv`
  - `csim_results.csv`
  - `fixed_pairs_100.csv`
  - ablation summary CSV
- notebook export or log snippets, if needed for verification

The normalization workflow copies supplied raw Kaggle exports into `logs/raw_kaggle_exports/`.
